# Connecting-Food-Test-Api

**Frist** Clone the repo
```sh
git clone https://github.com/MassDo/Connecting-Food-Test-Api.git
```
**Then** install dependencies and launch a virtual environment
```sh
cd Connecting-Food-Test-Api/
```
```sh
pipenv install
```
```sh
pipenv shell
```
**Finally** execute the migrations and start the server
```sh
cd api_test_project/
```
```sh
./manage.py migrate
```
```sh
./manage.py runserver
```
You can go to http://127.0.0.1:8000/ now !
